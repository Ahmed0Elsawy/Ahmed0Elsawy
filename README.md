## 🚀 Featured Projects

### 🏛️ PHARAOH-Guard — AI Crowd-Risk & Heritage Protection System
> *DEPI Graduation Project* · `Python` `Scikit-learn` `LangChain` `LangGraph` `FastAPI` `FAISS` `Groq LLM`

End-to-end decision-support system for crowd safety across **8 Egyptian heritage sites**: risk prediction → operational guidance → automated dispatch with audit logging.
- 🎯 4-class risk classifier: **96.6% accuracy / 0.93 macro-F1** on a 6-month time-based holdout, with **98% recall** on the rare Critical class (<1% base rate)
- 🤖 RAG + multi-agent architecture (analyst / historian / writer team) over a **5,852-document FAISS archive**, with a deterministic safety core — the LLM decides *when* to act, never *what* the safety actions are
- ⚡ Staged triage auto-closes **~85% of routine cases with zero LLM cost**, backed by a SQLite audit trail, live operations dashboard, and a 20-test suite

### 🤖 [SmartML — Intelligent AutoML Platform for Small Datasets](https://github.com/mostafataha12/smart-dataset-whisperer)

Non-technical users can upload data, configure models, and export deployment-ready artifacts.

### 📅 [Intelligent University Timetable Generator](https://github.com/Ahmed0Elsawy/University-Timetable-Scheduling-System)

### 🩺 [Melanoma Skin Cancer Detection](https://github.com/Ahmed0Elsawy/Skin-Cancer-Detection)
